### Acquire Dependencies using vcpkg
- Download [vcpkg](https://vcpkg.io/en/getting-started.html)
  - Install deps
    - `vcpkg install gtest --triplet=x64-windows`
  - Integrate install
    - `vcpkg integrate install`
